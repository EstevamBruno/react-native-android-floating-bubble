# react-native-android-floating-bubble

Android floating bubble overlay (Messenger-style) for React Native. iOS no-op.

## Installation


```sh
npm install react-native-android-floating-bubble
```


## Usage


```js
import { multiply } from 'react-native-android-floating-bubble';

// ...

const result = multiply(3, 7);
```
